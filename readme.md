# Crecaly - YouTube Revenue Calculator Extension

A lightweight, browser-based utility to calculate actual net income for creators. This repository contains the core mathematical logic and extension framework that powers **[Crecaly - Creator Income Calculator](https://crecaly.com)**.

## Overview
Unlike standard calculators, this extension factors in the complex variables that determine true creator revenue. It operates entirely locally within the browser to provide instant calculations for:
* **Niche-Specific RPMs:** Automatic adjustments for Finance, Tech, Gaming, Education, and Lifestyle content.
* **Geographic Audience Tiers:** Multipliers for Tier 1, Tier 2, and Tier 3 viewership locations.
* **Global Tax Deductions:** Instant net-income rendering after local tax percentages.

## Usage
This extension is designed to provide immediate revenue reality-checks directly from your browser toolbar. Enter your projected monthly views, select your content niche, set your audience geography, and instantly see your net earnings.
